Backend для проекта FrozenCondition.

Структура (первый черновик):
- `src/` — исходный код и пакеты для сбора/обработки данных.
- `data/` — сырые и промежуточные датасеты, загружаемые из CDAWEB и других источников.
- `tests/` — автотесты (пока пусто).

Рекомендуемая детализация:
- `src/frozen_condition/` — основной пакет проекта.
- `src/frozen_condition/io_cdaweb.py` — функции для работы с CDAWEB (аналог классов `CDAweb`, `RawData`, `DataDownloading` из ноутбука).
- `src/frozen_condition/processing.py` — функции предобработки и анализа данных.
- `src/frozen_condition/models/shue.py` — реализация модели Shue.
- `src/frozen_condition/config/` — параметры чтения/обработки (например, временные интервалы, спутники и т.п.).

Директория `data/` не должна содержать чувствительные данные в гите: лучше добавить крупные файлы в `.gitignore`, а в репозитории хранить только небольшие примеры.
