Frontend-часть проекта FrozenCondition.

Назначение:
- Jupyter-ноутбуки для визуализации данных и экспериментов;
- лёгкие скрипты для генерации графиков и фигур для статей.

Рекомендуемая структура:
- `notebooks/` — основные ноутбуки анализа и визуализации;
- `data/` — небольшие готовые датасеты (`.parquet`, `.csv` и др.) для быстрого построения графиков.

Тяжёлые исходные данные рекомендуется хранить в `backend/data` и не коммитить в репозиторий.
